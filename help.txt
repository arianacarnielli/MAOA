Utilisation :

MAOA [nom du fichier prp sans espaces] OPTIONS

OPTIONS :
  --mode [AB, AC, AH, EB, EC, E2] :
      Mode de résolution, peut être l'un des suivants :
        AB : mode approché de base
        AC : mode approché avec coupe
        AH : mode approché heuristique
        EB : mode exact (première formulation) de base
        EC : mode exact (première formulation) avec coupe (mode par defaut)
        E2 : mode exact (deuxieme formulation)

  --start [AB, AC, AH, None] : par défault, AH
  --verbose [0, 1, 2] : par défault 1
  --repeat_approx [int] : par défault 10
  --timeout [int] : par défault -1
  --heuristic_steps [int] : par défault 1000
  --tol_exact [double entre 0 et 1] : par défault 1e-4

MAOA fichier.prp --mode AB
